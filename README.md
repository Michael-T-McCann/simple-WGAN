# Simple Wasserstein GAN
Get a simple Wasserstein GAN up and running;
expand from there...

1. https://arxiv.org/abs/1701.07875

## One-time Setup
1. conda env create -f .\environment.yml
1. conda activate simple-WGAN

## Each-time Setup
1. git pull
1. conda env update -f .\environment.yml
1. conda activate simple-WGAN
